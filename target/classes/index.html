🚀 Bem-vindo ao Servidor HTTP em Java

Este servidor foi desenvolvido do zero, utilizando Java puro com suporte para rotas, middlewares e arquivos estáticos.

🛠 Funcionalidades atuais

  • Leitura de requisições HTTP
  • Resposta com status e corpo
  • Middlewares customizáveis (ex: Auth, CORS)
  • Mapeamento de rotas
  • Serviço de arquivos estáticos (HTML, CSS)

📁 Estrutura do projeto

/src
 ├── main/java
 │   ├── Config/
 │   ├── HTTP/
 │   ├── middleware/
 │   └── Main.java
/resources
 └── static/
     ├── index.html
     └── style.css
